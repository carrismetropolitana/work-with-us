# Decisões Técnicas

- Utilizei Fastify para o backend com rotas separadas e MongoDB via Docker.
- Utilizei Next.js 13 com app directory para o frontend.
- Persistência de favoritos com MongoDB, incluindo `createdAtUnix` e `operationalDate`.
- Implementação de FAVORITE/UNFAVORITE com fetch e tratamento de erros.
- CORS configurado para permitir acesso do frontend em localhost.
- Componentes TML UI utilizados: <Button> e <Title>.
- Código modular, organizado e legível.
